# Aegis UI

This part of the repo contains the code for the UI of Aegis. It is written in React with React Router in framework mode.

# 

# Running Locally

If you do not wish to use the compose script, and want to run the project locally on your machine, you will run the following command

```shell
npm run dev
```

# Testing

The UI has VItest installed to run unit tests. When in this directory, simply run the following command to run frontend tests

```shell
npm run test
```
